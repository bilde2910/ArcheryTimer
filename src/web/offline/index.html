Download the archery timer

The archery timer is automatically installed to your computer when you first run it, so there is no need to manually download the timer. If you want, you may give it a try - disconnect from the Internet, and click here to return to the timer. It should open, even if you aren't online.

Alternatively, you may use the Java offline version of the timer. Use of this version is discouraged, but if the standard version somehow doesn't work for you, or are concerned that it won't work when you need it, you can download it below.
